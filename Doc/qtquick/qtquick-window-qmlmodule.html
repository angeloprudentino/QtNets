Qt Quick Window QML Types

This QML module contains types for creating top-level windows and accessing screen information.

To use the types in this module, import the module with the following line:


  import QtQuick.Window 2.2

CloseEvent

Notification that a Window is about to be closed

Screen

Attached object provides information about the Screen an Item or Window is displayed on

Window

Creates a new top-level window
