Obsolete Members for QSplitter

The following members of class QSplitter are obsolete. They are provided to keep old source code working. We strongly advise against using them in new code.

Related Non-Members

(obsolete) QTextStream &operator<<(QTextStream &ts, const QSplitter &splitter)
(obsolete) QTextStream &operator>>(QTextStream &ts, QSplitter &splitter)

Related Non-Members

QTextStream &operator<<(QTextStream &ts, const QSplitter &splitter)

Use ts << splitter.saveState() instead.

QTextStream &operator>>(QTextStream &ts, QSplitter &splitter)

Use ts >> splitter.restoreState() instead.
