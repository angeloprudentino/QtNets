Embedded QWindow

This OS X-specific example shows how to create a view hierarchy containing a QWindow, with handling of Qt's mouse and keyboard events.

Running the Example

To run the example from Qt Creator, open the Welcome mode and select the example from Examples. For more information, visit Building and Running an Example.
