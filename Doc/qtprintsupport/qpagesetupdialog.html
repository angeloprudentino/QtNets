Detailed Description

The QPageSetupDialog class provides a configuration dialog for the page-related options on a printer.

On Windows and OS X the page setup dialog is implemented using the native page setup dialogs.

Note that on Windows and OS X custom paper sizes won't be reflected in the native page setup dialogs. Additionally, custom page margins set on a QPrinter won't show in the native OS X page setup dialog.

See also QPrinter and QPrintDialog.
