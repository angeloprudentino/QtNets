Qt Labs Calendar QML Types

The Qt Labs Calendar module provides QML types for creating calendars. These QML types work in conjunction with Qt Quick and Qt Quick Controls 2.

The QML types can be imported into your application using the following import statement in your .qml file:


  import Qt.labs.calendar 1.0

QML Types

Calendar

A calendar namespace

CalendarModel

A calendar model

DayOfWeekRow

A row of names for the days in a week

MonthGrid

A grid of days for a calendar month

WeekNumberColumn

A column of week numbers

Note: Types in the Qt.labs module are not guaranteed to remain compatible in future versions.
