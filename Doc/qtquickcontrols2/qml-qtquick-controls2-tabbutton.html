TabButton QML Type

A tab button control that can be found on a TabBar. More...

Detailed Description

TabButton is used in conjunction with a TabBar.


  TabBar {
      TabButton {
          text: qsTr("Home")
      }
      TabButton {
          text: qsTr("Discover")
      }
      TabButton {
          text: qsTr("Activity")
      }
  }

See also TabBar, Customizing TabButton, and Navigation Controls.
