Qt NFC QML Types

To use the classes of the module in your application, you need the following import statement in your .qml file:


  import QtNfc 5.2

NdefFilter

Represents a filtering constraint for NDEF message records

NdefMimeRecord

Represents an NFC MIME record

NdefRecord

Type represents a record in an NDEF message

NdefTextRecord

Represents an NFC RTD-Text NDEF record

NdefUriRecord

Represents an NFC RTD-URI NDEF record

NearField

Provides access to NDEF messages stored on NFC Forum tags
