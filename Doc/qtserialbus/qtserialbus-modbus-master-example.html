Modbus Master example

The example acts as Modbus master sending Modbus request via serial line and TCP respectively. The shown dialog allows the definition of standard requests and displays incoming responses.

Running the Example

To run the example from Qt Creator, open the Welcome mode and select the example from Examples. For more information, visit Building and Running an Example.

The example must be used in conjunction with the Modbus Slave example or another Mobus device which is either connected via TCP or Serial Bus.
